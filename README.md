# open-erp
Technologies and requirements
 - Java 1.8
 - Spring Boot 2.1.7
 - MAVEN 4.0.0
 - JSTL
 - JSON, XML (Jackson)
 - Tomcat 8 (embedded web server)
 - Hibernate
 - Lombok plugin
 - MySQL (does not depend on any rdbms)
 - Bootstrap 4
 - HTML#5
 - CSS
 
 package starter class: com.openerp.config.OpenERPApplication
